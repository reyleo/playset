Hello! This is a web application for playing famous SET game.

  • Play alone, with friends or family (up to 6 players supported)
  • Optimized for use with touchscreens
  • Add to home screen and enjoy playing at anytime!

SET is a real-time card game designed by Marsha Falco in 1974. The deck consists of 81 cards varying in four features: number (one, two, or three); symbol (diamond, squiggle, oval); shading (solid, striped, or open); and color (red, green, or purple).

How to play

The object of the game is to identify a Set of 3 cards from cards placed face up on the table. A SET consist of 3 cards in which each of the cards' features, looked at one-by-one, are the same one each card, or, are different on each card.
In other words: if 2 cards are the same and 1 card is different in any feature, then this is not a Set. A Set must be either all the same or all different in each individual feature
